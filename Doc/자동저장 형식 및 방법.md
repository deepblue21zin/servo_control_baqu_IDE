Windows PowerShell에서 아래 스크립트 실행
$port   = "COM5"   # 네 보드 COM 포트로 변경
$logDir = "C:\your_project\docs\measurements"
$ts     = Get-Date -Format "yyyy-MM-dd_HHmmss"
$log    = Join-Path $logDir "latency_$ts.csv"

New-Item -ItemType Directory -Force -Path $logDir | Out-Null

Start-Process "C:\Program Files\PuTTY\putty.exe" `
  -ArgumentList "-serial $port -sercfg 115200,8,n,1,N -sessionlog `"$log`" -logoverwrite"
PuTTY 창이 뜨면 바로 로그가 파일로 저장됨
펌웨어가 출력하는
LATENCY_BATCH_BEGIN,...
LATENCY_STAGE,...
LATENCY_BATCH_END,...
가 그대로 CSV 파일에 쌓임
